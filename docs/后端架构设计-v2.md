### Bounded Contexts

| Context | 职责 | 聚合根 | API前缀 |
|---------|------|--------|---------|
| **Shared Kernel** | 共享能力 | AuditLog, Attachment, Notification | `/api/auth`, `/api/attachments`, `/api/notifications` |
| **Organization** | 组织管理 | Organization, User, Role | `/api/organizations`, `/api/users`, `/api/roles` |
| **Strategy** | 战略管理 | AssessmentCycle, StrategicPlan, StrategicIndicator | `/api/cycles`, `/api/plans`, `/api/indicators` |
| **Execution** | 执行管理 | Task, ProgressReport | `/api/tasks`, `/api/reports` |
| **Workflow** | 工作流引擎 | WorkflowDefinition, WorkflowInstance | `/api/workflows`, `/api/approvals` |
| **Analytics** | 数据分析 | (无聚合根，只读服务) | `/api/analytics` |

## 设计原则

### 1. 领域驱动设计 (DDD)

**核心概念**：

```
Bounded Context (限界上下文)
    │
    ├── Aggregate (聚合)
    │     └── Aggregate Root (聚合根)
    │     └── Entity (实体)
    │     └── Value Object (值对象)
    │
    ├── Repository (仓储)
    │     └── 只通过聚合根访问数据
    │
    ├── Domain Service (领域服务)
    │     └── 复杂业务逻辑
    │
    └── Domain Event (领域事件)
          └── 跨聚合通信
```

**示例 - Strategy Context**：

```java
// 聚合根
@Entity
public class StrategicPlan {
    @Id
    private String id;

    private String name;
    private PlanStatus status;

    // 实体（一对多）
    @OneToMany(mappedBy = "plan")
    private List<StrategicIndicator> indicators;

    // 值对象
    private PlanPeriod period;

    // 业务方法
    public void submitForApproval() {
        if (this.status != PlanStatus.DRAFT) {
            throw new IllegalStateException("只有草稿状态的计划可以提交审批");
        }
        this.status = PlanStatus.PENDING_APPROVAL;
    }
}

// 实体
@Entity
public class StrategicIndicator {
    @Id
    private String id;

    @ManyToOne
    private StrategicPlan plan;

    private String name;
    private IndicatorStatus status;
}

// 值对象（不可变）
@Embeddable
public class PlanPeriod {
    private LocalDate startDate;
    private LocalDate endDate;

    // 业务逻辑
    public boolean isActive() {
        return LocalDate.now().isAfter(startDate)
            && LocalDate.now().isBefore(endDate);
    }
}
```

### 2. 分层架构

每个 Bounded Context 内部分为四层：

```
interfaces/     # 接口层
    │          - REST Controllers
    │          - DTOs
    │          - Request/Response 映射
    │
application/    # 应用层
    │          - Application Services
    │          - DTOs
    │          - 用例编排
    │
domain/         # 领域层
    │          - Aggregates (聚合根)
    │          - Entities (实体)
    │          - Value Objects (值对象)
    │          - Domain Services (领域服务)
    │          - Repository Interfaces (仓储接口)
    │          - Domain Events (领域事件)
    │
infrastructure/ # 基础设施层
               - Repository Implementations (仓储实现)
               - mappers (对象映射)
               - external service clients
```

**依赖规则**：

```
interfaces → application → domain
infrastructure → domain
```

**示例代码**：

```java
// domain/model/StrategicPlan.java - 聚合根
@Entity
public class StrategicPlan {
    // 领域逻辑
    public void submitForApproval() {
        validateCanSubmit();
        this.status = PlanStatus.PENDING_APPROVAL;
        // 发布领域事件
        registerEvent(new PlanSubmittedEvent(this.id));
    }
}

// domain/repository/PlanRepository.java - 仓储接口
public interface PlanRepository {
    Optional<Plan> findById(String id);
    void save(Plan plan);
    List<Plan> findByCycle(String cycleId);
}

// application/service/PlanApplicationService.java - 应用服务
@Service
public class PlanApplicationService {
    private final PlanRepository planRepository;
    private final WorkflowService workflowService;

    @Transactional
    public void submitPlan(String planId) {
        // 获取聚合根
        Plan plan = planRepository.findById(planId)
            .orElseThrow(() -> new PlanNotFoundException(planId));

        // 执行业务逻辑
        plan.submitForApproval();

        // 保存
        planRepository.save(plan);

        // 启动工作流
        workflowService.startWorkflow("PLAN_APPROVAL", planId);
    }
}

// interfaces/rest/PlanController.java - 控制器
@RestController
@RequestMapping("/api/plans")
public class PlanController {
    private final PlanApplicationService planService;

    @PostMapping("/{id}/submit-approval")
    public ResponseEntity<Void> submitApproval(@PathVariable String id) {
        planService.submitPlan(id);
        return ResponseEntity.ok().build();
    }
}

// infrastructure/persistence/PlanRepositoryImpl.java - 仓储实现
@Repository
public class PlanRepositoryImpl implements PlanRepository {
    @PersistenceContext
    private EntityManager em;

    @Override
    public Optional<Plan> findById(String id) {
        return Optional.ofNullable(em.find(Plan.class, id));
    }

    @Override
    public void save(Plan plan) {
        if (plan.getId() == null) {
            em.persist(plan);
        } else {
            em.merge(plan);
        }
    }
}
```

### 3. 依赖倒置

**原则**：高层模块不应依赖低层模块，两者都应依赖抽象。

**实现**：
- 定义接口在 domain 层
- 实现在 infrastructure 层
- 应用层依赖 domain 接口

```java
@Entity
@Table(name = "strategic_plan")
public class StrategicPlan {

    @Id
    @GeneratedValue(strategy = GenerationType.UUID)
    private String id;

    private String name;
    private String code;
    private String description;

    @Enumerated(EnumType.STRING)
    private PlanStatus status;

    @Embedded
    private PlanPeriod period;

    @ManyToOne
    private AssessmentCycle cycle;

    @ManyToOne
    private Organization organization;

    @OneToMany(mappedBy = "plan", cascade = CascadeType.ALL)
    private List<StrategicIndicator> indicators = new ArrayList<>();

    private String createdBy;
    private LocalDateTime createdAt;
    private LocalDateTime updatedAt;

    // ========== 业务方法 ==========

    public void submitForApproval() {
        if (this.status != PlanStatus.DRAFT) {
            throw new IllegalStateException("只有草稿状态的计划可以提交审批");
        }
        if (this.indicators.isEmpty()) {
            throw new IllegalStateException("计划必须包含至少一个指标");
        }
        this.status = PlanStatus.PENDING_APPROVAL;
        this.registerEvent(new PlanSubmittedEvent(this.id));
    }

    public void approve() {
        if (this.status != PlanStatus.PENDING_APPROVAL) {
            throw new IllegalStateException("只有待审批状态的计划可以被批准");
        }
        this.status = PlanStatus.APPROVED;
        this.registerEvent(new PlanApprovedEvent(this.id));
    }

    public void reject(String reason) {
        if (this.status != PlanStatus.PENDING_APPROVAL) {
            throw new IllegalStateException("只有待审批状态的计划可以被拒绝");
        }
        this.status = PlanStatus.DRAFT;
        this.registerEvent(new PlanRejectedEvent(this.id, reason));
    }

    public void addIndicator(StrategicIndicator indicator) {
        if (this.status != PlanStatus.DRAFT) {
            throw new IllegalStateException("只有草稿状态的计划可以添加指标");
        }
        indicator.setPlan(this);
        this.indicators.add(indicator);
    }

    public void removeIndicator(String indicatorId) {
        if (this.status != PlanStatus.DRAFT) {
            throw new IllegalStateException("只有草稿状态的计划可以删除指标");
        }
        this.indicators.removeIf(i -> i.getId().equals(indicatorId));
    }

    // 领域事件
    private List<DomainEvent> events = new ArrayList<>();

    private void registerEvent(DomainEvent event) {
        this.events.add(event);
    }

    public List<DomainEvent> getUncommittedEvents() {
        return Collections.unmodifiableList(events);
    }

    public void markEventsAsCommitted() {
        events.clear();
    }
}
```

```java
@Entity
@Table(name = "strategic_indicator")

    @Id
    @GeneratedValue(strategy = GenerationType.UUID)

    private String name;
    private String code;
    private String description;

    private LocalDateTime updatedAt;

    // ========== 业务方法 ==========

    public void distributeTo(Organization organization) {
        if (this.status != IndicatorStatus.APPROVED) {
            throw new IllegalStateException("只有已批准的指标可以下发");
        }
        this.targetOrganization = organization;
        this.status = IndicatorStatus.DISTRIBUTED;
        this.registerEvent(new IndicatorDistributedEvent(this.id, organization.getId()));
    }

    public void withdraw() {
        if (this.status != IndicatorStatus.DISTRIBUTED) {
            throw new IllegalStateException("只有已下发的指标可以撤回");
        }
        this.targetOrganization = null;
        this.status = IndicatorStatus.APPROVED;
        this.registerEvent(new IndicatorWithdrawnEvent(this.id));
    }

    public void updateProgress(Double currentValue) {
        if (this.status != IndicatorStatus.DISTRIBUTED) {
            throw new IllegalStateException("只有已下发的指标可以更新进度");
        }
        this.target.setCurrentValue(currentValue);

        if (this.target.isAchieved()) {
            this.registerEvent(new IndicatorAchievedEvent(this.id));
        } else if (this.target.isAtRisk()) {
            this.registerEvent(new IndicatorAtRiskEvent(this.id));
        }
    }

    // 领域事件

    private void registerEvent(DomainEvent event) {
        this.events.add(event);
    }

    public List<DomainEvent> getUncommittedEvents() {

    }
```

```java
@Embeddable
public class PlanPeriod {

    @Column(name = "start_date")
    private LocalDate startDate;

    @Column(name = "end_date")
    private LocalDate endDate;

    public PlanPeriod(LocalDate startDate, LocalDate endDate) {
        if (startDate.isAfter(endDate)) {
            throw new IllegalArgumentException("开始日期不能晚于结束日期");
        }
        this.startDate = startDate;
        this.endDate = endDate;
    }

    public boolean isActive() {
        LocalDate now = LocalDate.now();
        return !now.isBefore(startDate) && !now.isAfter(endDate);
    }

    public boolean isFuture() {
        return LocalDate.now().isBefore(startDate);
    }

    public boolean isPast() {
        return LocalDate.now().isAfter(endDate);
    }

    public Duration getDuration() {
        return Duration.between(startDate, endDate);
    }

    // 不可变性
    public PlanPeriod withEndDate(LocalDate newEndDate) {
        return new PlanPeriod(this.startDate, newEndDate);
    }

    // equals, hashCode based on values
    @Override
    public boolean equals(Object o) {
        if (this == o) return true;
        if (o == null || getClass() != o.getClass()) return false;
        PlanPeriod that = (PlanPeriod) o;
        return Objects.equals(startDate, that.startDate) &&
               Objects.equals(endDate, that.endDate);
    }

    @Override
    public int hashCode() {
        return Objects.hash(startDate, endDate);
    }
}
```

#### IndicatorTarget (指标目标)

```java
@Embeddable
public class IndicatorTarget {

    @Column(name = "target_value")
    private Double targetValue;

    @Column(name = "current_value")
    private Double currentValue = 0.0;

    @Column(name = "unit")
    private String unit;

    @Column(name = "min_value")
    private Double minValue;

    @Column(name = "max_value")
    private Double maxValue;

    public IndicatorTarget(Double targetValue, String unit) {
        this.targetValue = targetValue;
        this.unit = unit;
    }

    public Double getProgressPercent() {
        if (targetValue == 0) return 0.0;
        return (currentValue / targetValue) * 100;
    }

    public boolean isAchieved() {
        return currentValue >= targetValue;
    }

    public boolean isAtRisk() {
        if (minValue == null) return false;
        return currentValue < minValue;
    }

    public boolean isOnTrack() {
        return !isAchieved() && !isAtRisk();
    }

    public void updateCurrent(Double value) {
        if (minValue != null && value < minValue) {
            throw new IllegalArgumentException("当前值不能低于最小值");
        }
        if (maxValue != null && value > maxValue) {
            throw new IllegalArgumentException("当前值不能高于最大值");
        }
        this.currentValue = value;
    }
}
```

```java
// domain/repository/IndicatorRepository.java
public interface IndicatorRepository {
    Optional<StrategicIndicator> findById(String id);
    List<StrategicIndicator> findByPlanId(String planId);
    List<StrategicIndicator> findByStatus(IndicatorStatus status);
    void save(StrategicIndicator indicator);
    void delete(String id);
}

// infrastructure/persistence/IndicatorRepositoryImpl.java
@Repository
public class IndicatorRepositoryImpl implements IndicatorRepository {

    @PersistenceContext
    private EntityManager em;

    @Override
    public Optional<StrategicIndicator> findById(String id) {
        return Optional.ofNullable(
            em.createQuery(
                "SELECT i FROM StrategicIndicator i LEFT JOIN FETCH i.plan WHERE i.id = :id",
                StrategicIndicator.class
            )
            .setParameter("id", id)
            .getSingleResult()
        );
    }

    @Override
    public List<StrategicIndicator> findByPlanId(String planId) {
        return em.createQuery(
            "SELECT i FROM StrategicIndicator i WHERE i.plan.id = :planId",
            StrategicIndicator.class
        )
        .setParameter("planId", planId)
        .getResultList();
    }

    @Override
    public void save(StrategicIndicator indicator) {
        if (indicator.getId() == null) {
            em.persist(indicator);
        } else {
            em.merge(indicator);
        }
    }

    @Override
    public void delete(String id) {
        StrategicIndicator indicator = findById(id)
            .orElseThrow(() -> new IndicatorNotFoundException(id));
        em.remove(indicator);
    }
}
```
