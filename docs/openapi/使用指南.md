## 🚀 前端如何使用

### 方法1: 导入Postman（推荐）

1. 打开Postman
2. 点击 `Import` → `File`
3. 选择 `openapi-full.json`
4. 自动创建所有API请求和环境变量

### 方法2: 在线Swagger UI

启动后端服务后访问：
```
http://localhost:8080/api/swagger-ui.html
```

### 方法3. 生成前端API客户端

#### TypeScript/Axios
```bash
npm install -g @openapitools/openapi-generator-cli
openapi-generator-cli generate \
  -i docs/openapi/openapi-full.json \
  -g typescript-axios \
  -o src/api/generated
```

#### JavaScript/Fetch
```bash
openapi-generator-cli generate \
  -i docs/openapi/openapi-full.json \
  -g javascript \
  -o src/api/generated
```

## 🔑 认证示例

```javascript
// 1. 登录获取Token
const response = await fetch('http://localhost:8081/api/v1/auth/login', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    username: 'admin',
    password: 'admin123'
  })
});

const { data } = await response.json();
const token = data.access_token;

// 2. 使用Token访问API
const apiResponse = await fetch('http://localhost:8080/api/v1/indicators', {
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  }
});
```

## 📝 核心API端点

### 指标管理
- `GET /api/v1/indicators` - 查询指标列表
- `POST /api/v1/indicators` - 创建指标
- `POST /api/v1/indicators/{id}/distribute` - 下发指标
- `POST /api/v1/indicators/{id}/submit-approval` - 提交审批

### 审批流程
- `POST /api/v1/approval/instances/{instanceId}/approve` - 审批通过
- `POST /api/v1/approval/instances/{instanceId}/reject` - 审批驳回
- `GET /api/v1/approval/instances/my-pending` - 我的待办

### 进度填报
- `POST /api/v1/reports` - 创建报告
- `POST /api/v1/reports/{id}/submit` - 提交报告
- `GET /api/v1/reports/indicator/{indicatorId}` - 查询指标报告
