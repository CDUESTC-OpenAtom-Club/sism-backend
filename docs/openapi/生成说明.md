# OpenAPI文档生成说明

## ⚠️ 端口说明

- **后端服务端口**: 8080（根据.env配置）
- **数据库端口**: 8386（PostgreSQL）
- **Swagger UI访问**: http://localhost:8080/api/swagger-ui.html
- **OpenAPI JSON**: http://localhost:8080/api-docs

## 🔧 如何重新生成

如果API有更新，按以下步骤重新生成：

1. **启动后端服务**
```bash
cd sism-main
mvn spring-boot:run -Dspring-boot.run.profiles=dev
```

2. **等待启动完成**（约30秒）

3. **生成OpenAPI文档**
```bash
curl -s http://localhost:8080/api-docs | jq '.' > docs/openapi/openapi-full.json
```

4. **验证生成**
```bash
jq '.openapi, .info.version, .paths | length' docs/openapi/openapi-full.json
```

应该输出：
```
"3.0.1"
"1.0.0"
181  # API端点数量
```
