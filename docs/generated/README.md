# Generated 文档产物目录

本目录用于存放**自动生成**的后端分析报告、分片清单和索引文件，避免与手写设计文档混放。

**默认规则已正式确定：后续自动生成的后端设计、排查、优化、汇报、清单类文档，统一优先落到 `docs/generated/`。**

详细约定见：`CONVENTION.md`

## 存储约定

- **报告根目录**：`performance-reports/<module>/<date>/`
- **正文分片**：`chunks/0001-*.md` 到 `chunks/NNNN-*.md`
- **元信息**：`report.meta.json`
- **清单索引**：`manifest.json`
- **重建方式**：使用 `scripts/report-chunks/render-report.js`

## 设计目标

- **有序性**：依赖四位序号命名和 `manifest.json` 的 `seq` 字段保持顺序。
- **可检索性**：清单中保存标题、章节、关键词和摘要预览，便于按模块、日期、主题检索。
- **空间利用**：默认只保存分片正文和索引，不额外保存完整拼接副本；需要查看完整报告时再按需重建。
- **可扩展性**：新增模块报告时复用相同目录结构和脚本即可。
