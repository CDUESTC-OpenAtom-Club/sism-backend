# SISM - 战略指标管理系统

Strategic Indicator Management System (SISM) 是一个面向高校的战略任务与指标管理平台。

## 核心功能

- **数据看板**: 可视化展示战略任务完成情况、指标进度、部门对比
- **指标管理**: 指标的创建、分配、填报、审批全流程
- **里程碑跟踪**: 按时间节点跟踪指标完成进度
- **多角色权限**: 三级组织架构（战略发展部 → 职能部门 → 二级学院）
- **审计日志**: 完整记录所有操作历史

## 用户角色

| 角色 | 说明 |
|------|------|
| `strategic_dept` | 战略发展部 - 系统管理员，负责指标下发 |
| `functional_dept` | 职能部门 - 中层管理，负责指标分解和审批 |
| `secondary_college` | 二级学院 - 执行层，负责指标填报 |

## 业务流程

1. 战略发展部创建战略任务和指标
2. 指标下发至职能部门或二级学院
3. 执行部门按里程碑填报进度
4. 上级部门审批进度报告
5. 系统自动生成预警和统计报表
